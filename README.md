## Project Description
Title: Personal Web Portfolio

# Technologies Used: Next.js, Tailwind CSS, EmailJS

# Overview: 
This project involves creating a dynamic and visually appealing personal portfolio website using Next.js, Tailwind CSS, and EmailJS. The portfolio will showcase various web development skills, projects, and contact information.

# Key Features:

 - Next.js: Utilized for server-side rendering and static site generation, ensuring fast load times and SEO optimization. The project includes multiple pages like Home, Skills & Experience, Projects, and Contact.

 - Tailwind CSS: Employed for styling the website with utility-first CSS classes, allowing for responsive design and custom styling without writing custom CSS for every component.

 - EmailJS: Integrated to handle contact form submissions, enabling users to send emails directly from the portfolio without exposing sensitive email credentials. This feature includes form validation and feedback messages for successful or failed submissions.

# Functionality:

 - Home Page: Displays a welcome message and an overview of the skills and projects.

 - Skills & Experience: Highlights various technical skills and experiences with icons and descriptions.

 - Projects: Showcases a list of completed projects with descriptions and links to view more details.

 - Contact Page: Provides a contact form that uses EmailJS to send emails directly to the user's email address. Includes form validation to ensure that all required fields are filled out correctly.
